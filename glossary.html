Glossary

Hyperlink
A hyperlink creates a clickable link that can point to a file, document, page and just about anything else that lives on the web
Debugging
Debugging is the process of finding and fixing errors within code that may stop it from functioning properly
Formatting
Formatting is the process of tidying up code so it not only works correctly but is easier to read and understand
Nesting
Nesting is when an element is placed inside another element, it is probably one of the most used tools in coding in my experience
Empty elements
Empty elements are elements that exist with only one tag, there is no closing tag with an empty element
